Er zijn verschillende selectors, bijvoorbeeld:

  • type selectors

    De CSS-type selector komt overeen met elementen op knooppuntnaam. Met andere woorden, het selecteert alle elementen van het gegeven type binnen een document (dit is de voordeel van dit selector).

    Syntax : element { style properties }

    Voorbeelden:

    1. span {
      background-color: skyblue; }
    2. h1 {
      color: blue;
      }
    3. @namespace example url(http://www.example.com);
      example|h1 { color: blue }

  • descendant selectors

    De descendant combinator - meestal weergegeven door een enkele spatie () - combineert twee selectors zodat elementen die overeenkomen met de tweede selector worden geselecteerd als ze een voorouder (ouder, ouder van ouder, ouder van ouder, enz.) - element hebben dat overeenkomt met de eerste selector. Selectors die een afstammingscombinator gebruiken, worden descendant selectors genoemd.

    • Voorbeelden:

    1. li {
      list-style-type: disc;
      }
      li li {
      list-style-type: circle;
      }

    2. /* List items that are descendants of the "my-things" list */
      ul.my-things li {
      margin: 2em;
      }

    3. div p {
      background-color: yellow;
      }

    Een vordeel van descendant selectors is stylen zonder andere klassen of ids maken.

  • class selectors

    De CSS class selector komt overeen met elementen op basis van de inhoud van hun class attribuut.

    Syntax:
    .class_name { style properties }
    or
    [class~=class_name] { style properties }

    Voorbeelden:

    1. /* All elements with class="spacious" */
      .spacious {
      margin: 2em;
      }
      /* All <li> elements with class="spacious" */
      li.spacious {
      margin: 2em;
      }
      /* All <li> elements with a class list that includes both "spacious" and "elegant" */
      /* For example, class="elegant retro spacious" */
      li.spacious.elegant {
      margin: 2em;
      }
    Door een klasse aan een element toe te wijzen, worden de menselijke lezers van de code verteld dat die elementen tot een bepaalde groep behoren en dat de groep waartoe ze behoren relatief dezelfde stijlen zou moeten hebben. Dit maakt het verwijzen, onderhouden en bijwerken van CSS-stijl veel eenvoudiger en sneller.
